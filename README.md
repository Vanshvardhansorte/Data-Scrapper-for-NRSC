## 🚀 **Project Overview**

The ** Data Pipeline** is a comprehensive intelligence tool designed to bridge the gap between raw web data and structured analysis. Originally conceptualized for high-integrity environments, this **universal scraper** automates the **Extraction, Structuring, and Cleaning** of information from **any public URL**.

By utilizing a **dual-engine approach**:
- **Selenium** for dynamic JS rendering
- **BeautifulSoup** for parsing

The pipeline ensures **100% data capture accuracy** across diverse web architectures.


## ✨ **Key Features**

The pipeline operates in **six distinct operational modes**:

| Mode | Description | Use Case |
|------|-------------|----------|
| 🔍 **CSS Selector** | Target specific HTML elements (classes, IDs, tags). Use `"everything"` for broad semantic sweep. | Precise element extraction |
| 📁 **File Discovery** | Auto-identify download links for extensions (.pdf, .docx, .zip) | Document harvesting |
| 📊 **Metadata Scraper** | Extract SEO intelligence (titles, meta-descriptions) | Competitive analysis |
| 🔎 **Keyword Search** | Scan content segments containing specific terms | Targeted intelligence |
| 🖼️ **Image Scraper** | Capture image URLs + Alt-text descriptions | Visual asset collection |
| 📈 **Table Extractor** | Convert HTML tables to structured datasets | Data normalization |

## 🛠️ Tech Stack

- **Frontend:** Streamlit (Responsive Dashboard)  
- **Engine:** Selenium WebDriver (Headless Chrome)  
- **Parser:** BeautifulSoup4  
- **Data Processing:** Pandas (Normalization & De-duplication)  
- **Reporting:** FPDF (PDF Intelligence Reports)  
- **Deployment:** Streamlit Cloud
